ChromeKit — Chrome 확장 프로그램 큐레이션

ChromeKit은 수천 개의 Chrome 확장 프로그램 중에서 실제로 유용하고 검증된 것만을 엄선해 카테고리별로 정리한 큐레이션 서비스입니다. 생산성 향상을 위한 Notion Web Clipper, Dark Reader, OneTab부터 보안을 위한 uBlock Origin, Bitwarden까지 다양한 확장 프로그램을 한 곳에서 바로 설치할 수 있습니다.

개발자라면 React DevTools, Redux DevTools, Wappalyzer, Octotree 같은 필수 개발자 확장 프로그램을 놓치지 마세요. 디자이너에게는 ColorZilla, WhatFont, Eye Dropper, PerfectPixel이 작업 효율을 크게 높여줍니다. 수십 개의 탭을 관리해야 한다면 Tab Manager Plus, Workona, Session Buddy로 브라우저를 정돈하세요.

온라인 쇼핑 시에는 Honey, Rakuten, Capital One Shopping으로 쿠폰 코드를 자동 적용하고 최저가를 비교할 수 있습니다. 학습과 리서치에는 Pocket, Mercury Reader, Wikiwand, Hypothesis를 활용해 콘텐츠를 효율적으로 저장하고 분석하세요.

ChromeKit은 광고성·스팸 확장 프로그램 없이 실제로 사용해보고 추천하는 확장만을 수록합니다. 크롬 웹 스토어 공식 링크를 통해 안전하게 설치할 수 있으며, 신뢰할 수 있는 개발사의 확장만 포함합니다.
